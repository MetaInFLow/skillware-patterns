# Pipes and Filters / 管道-过滤器模式

This standalone record transfers the Pattern-Oriented Software Architecture
Pipes and Filters pattern to a Support Ticket Triage Skillware Unit. It is an
architectural pattern, explicitly not a Gang of Four pattern.

The constructive sample contains one Data Source, five independent Filter
Skills, a versioned Pipe contract, one Data Sink, deterministic fixtures, and a
standard-library executable oracle. Start with [`definition.md`](definition.md)
and [`sample/README.md`](sample/README.md).
