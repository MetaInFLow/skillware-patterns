# Why this is not Template Method

The domain-owned checklists replace the whole algorithm. Healthcare drafts
before extraction, and Finance may omit the mandatory quality check. A caller
cannot rely on one skeleton, and there is no primitive operation with a bounded
common contract.

This may be configurable workflow routing or whole-algorithm Strategy-like
selection, but it is not Template Method. Restoring the pattern requires the
root AbstractClass to own the invariant order and letting each ConcreteClass
implement only the declared `apply-domain-hook` operation.
